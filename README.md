
# Nuxt 實戰 - Vue 工程師的必修課 作業

## 專案說明
很顯然是用來交作業的。

## 環境需求
- Node.js 18.0 或更高版本
- npm 9.0 或更高版本
- Vue 3
- Nuxt 3.13+

## 安裝說明
```bash
# 安裝相依套件
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build
```
